## Requirements

- Node.js 22 or later
- One or more AI coding assistants installed (Claude Code, Gemini CLI, etc.)

## Install via npm

```bash
npm install -g argusdev
```

## Run without installing

```bash
npx argusdev
```

## Verify installation

```bash
argusdev --version
```
